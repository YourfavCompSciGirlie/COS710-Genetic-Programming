Symbolic regression of UK residential electricity load using tree-based Genetic Programming. The GP evolves closed-form arithmetic expressions that predict electricity consumption at time $t$ from the 7 most recent observed values (autoregressive window).

---

## Project Structure

```
Assignment 1/
├── src/
│   ├── Main.java               # Entry point, GP configuration, output
│   ├── GeneticProgramming.java # GP engine (selection, crossover, mutation, elitism)
│   ├── Tree.java               # Expression tree (evaluate, copy, grow, trim)
│   ├── Node.java               # Tree node (label, arity, children, parent)
│   └── DatasetLoader.java      # CSV loading, windowing, train/test split
├── documentation/
│   └── Residential_Energy_Dataset_UK- 2014-2020.csv
├── out/                        # Compiled .class files (generated by javac)
├── GP_Assignment1.jar          # Executable JAR (pre-built)
├── report.md                   # Assignment report
└── README.md
```

## Quick Start (JAR — recommended)

Run from the `Assignment 1/` directory:

```bash
java -jar GP_Assignment1.jar
```

The program runs 10 independent GP runs and prints per-run MSE, the best evolved expression, hits/success predicate summary, and per-generation statistics for the best run.

---

## Compile and Run from Source

From the `Assignment 1/` directory:

```bash
# Compile
javac -d out src/*.java

# Run
java -cp out Main
```
